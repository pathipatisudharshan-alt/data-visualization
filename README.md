# data-visualization
# 🚢 Titanic EDA — Exploratory Data Analysis

## 📌 Objective
Analyze the Titanic dataset to find patterns in survival using Python visualization libraries.

## 📊 Dataset
- **Source:** [Kaggle Titanic Dataset](https://www.kaggle.com/competitions/titanic)
- **Rows:** 891 passengers · **Columns:** 12

## 🔍 Key Findings
- Only **38%** of passengers survived
- **Females survived more** than males
- **Higher fare = better survival rate**
- Most passengers were **20–35 years old**
- **Class 1** passengers had highest survival rate

## 📈 Charts Created
- Survival Count (Bar chart)
- Survival by Gender (Count plot)
- Age Distribution (Histogram + KDE)
- Fare vs Age (Scatter plot)
- Correlation Heatmap
- Pairplot (Age, Fare, Pclass)

## 🛠 Libraries Used
- Pandas · NumPy · Matplotlib · Seaborn
